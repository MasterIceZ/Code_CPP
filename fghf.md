### list file
```shell
ls 
```
Open Directory
```shell
cd directory_name
```
Chmod
```shell
chomod 000-111/-x/x z.txt
```
Create Directory
```shell
mkdir make directory
```
Open Vim with file
```shell
vi file_name 
```
Compile with C++14
```shell
g++ -std=c++14 -O2 program_name.cpp -o execute_name
```
Run program with stdin
```shell
./program< input.txt
```
Run program with stdin and make output as a file
```shell
./program < input.txt > output.txt 
```
